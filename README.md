\# SkillBridge CV Analyzer



NLP-powered semantic CV-to-Job matching that surfaces hidden gem candidates.



\## Problem

ATS systems reject great candidates using keyword matching.

Career-switchers and bootcamp grads get auto-rejected unfairly.



\## Solution

Uses sentence transformers + NLP to semantically match CVs to job descriptions

and score candidates with a transparent Hidden Gem Score.



\## Quickstart

pip install -r requirements.txt

python -m src.main --cv data/sample\_cvs/example.txt --jd data/sample\_jds/example.txt
